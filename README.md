# 26-1exam-security

## 必要なもの
- Webブラウザ（Chrome, Firefoxなど）
- （任意）ローカルで簡単に閲覧するためのHTTPサーバ（下記コマンド参照）

## 使い方（手順）
1. リポジトリをクローンまたはダウンロードします。
2. ファイルをブラウザで開くか、ローカルサーバを立てて閲覧します。

ブラウザで直接開く場合は、リポジトリ内の該当HTMLファイルをダブルクリックしてください。模擬試験は次のフォルダにあります（英語名にコピー済み）。

- `mock_exam_1/`
- `mock_exam_2/`
- `mock_exam_3/`

それぞれのフォルダ内にある`test.html`が問題、`answer.html`が解答、`intent.html`が解説（意図）です。

## ローカルサーバで閲覧する方法（推奨）
ターミナルでリポジトリのルートに移動し、次のコマンドを実行します。

```bash
python3 -m http.server 8000
```

ブラウザで以下にアクセスします（ポートは上のコマンドの数字に合わせてください）：

http://localhost:8000/mock_exam_1/test.html

（他の回を見る場合はパスの数字を変更してください）

## 学習の進め方（大学生向けの提案）
-- まずは`00_overview.html`で構成を確認する。
-- 各章（`01_security_basics.html`〜`04_organizational_security_management.html`）を順に読み、重要語句をノートする。
- 模擬試験の`test.html`を時間を計って解く→`answer.html`で採点→`intent.html`で解説を確認。
- 間違えた問題は関連する章に戻り、理解を深める。

## カスタマイズ
